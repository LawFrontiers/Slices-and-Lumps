# Slices-and-Lumps《片与块》翻译协作平台

## （一）讨论方式
### 1. 通过Discussions页面进行交流  
    > 轻量级排版，可以熟悉一下markdown语法：https://markdown.com.cn/basic-syntax/  
### 2. 通过New Discussion创建讨论区
### 3. 通过Add a comment提出议题
### 4. 通过Write a reply参与讨论
### 5. 在Watch中选择提醒方式：确保更新能够在注册邮箱中即时收到讨论内容（`勾选Discussions`）  
![image](https://github.com/LegalGPT/Tuesday-Slices-and-Lumps/assets/85102787/331326f4-4e24-4d57-a6f5-47611e3c6e65)

## （二）讨论内容
### 1. 每章一个讨论区（discussion）
- 特定章节已有讨论区（discussion）：请在该讨论区提出议题和参与讨论
- 特定章节尚无讨论区（discussion）：通过New Discussion，为该章节创建讨论区
### 2. 在相关讨论区中（discussion）中提出新的议题（comment）
- 包括但不限于：术语、语句、段落等内容
- 建议先提供所在**段落**的全段原文，再针对特定内容提出观点
- 如果讨论区中已有相应议题，建议通过reply参与讨论，尽可能不要通过comment创建重复的议题
### 3. 在相关议题（comment）中提出新的观点
- 通过reply参与讨论，建议对于之前的观点给予回应，表明立场并给出理由
- 如果能够联系全书中的相关内容，将会更有说服力
- 设计相关内容，提供原文会很有帮助
